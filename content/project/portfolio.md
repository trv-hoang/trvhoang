## Overview

My personal portfolio and blog, built from scratch with Hugo and Tailwind CSS. Deployed on Cloudflare Pages with D1 and R2 for edge-optimized performance.

### Key Features

- **Static Site Generation** with Hugo for blazing fast load times
- **Tailwind CSS v4** for a custom, responsive design system
- **Cloudflare Pages** deployment with global CDN distribution
- **Dark/Light Mode** with smooth transitions and persistent preference
- **Blog System** with tag filtering and markdown-powered content
- **Interactive Map** page for showcasing locations

### My Responsibilities

- Designed and implemented the entire UI/UX from scratch
- Built reusable Hugo partials and layout templates
- Configured Cloudflare Pages, D1 database, and R2 storage
- Implemented SEO optimizations and Open Graph metadata
