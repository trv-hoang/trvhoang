## Overview

An internal CRM system built for 1Matrix to manage customer relationships, track interactions, and streamline business workflows. Built with a modern React frontend and Cloudflare-powered backend.

### Key Features

- **Customer Management** — centralized dashboard for managing client data and interactions
- **Activity Tracking** — logs and tracks all customer touchpoints and communications
- **Cloudflare Workers** backend for serverless API endpoints
- **Role-based Access Control** for team members with different permission levels

### My Responsibilities

- Built the React frontend with TypeScript for type-safe component development
- Designed and implemented the customer dashboard UI
- Integrated with Cloudflare Workers API endpoints
- Implemented responsive layouts for desktop and mobile usage
